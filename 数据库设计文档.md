# 数据库设计文档

## 概述
本项目使用SQLite数据库，采用SQLAlchemy ORM框架进行数据库操作。

## 数据库表结构

### 1. categories（科目分类表）
| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| id | INTEGER | PRIMARY KEY | 主键ID |
| name | VARCHAR(50) | UNIQUE, NOT NULL | 科目名称 |
| description | VARCHAR(200) | | 科目描述 |
| created_at | DATETIME | DEFAULT NOW | 创建时间 |

### 2. knowledge_points（知识点表）
| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| id | INTEGER | PRIMARY KEY | 主键ID |
| name | VARCHAR(100) | NOT NULL | 知识点名称 |
| category_id | INTEGER | FOREIGN KEY | 所属科目ID |
| description | VARCHAR(500) | | 知识点描述 |
| created_at | DATETIME | DEFAULT NOW | 创建时间 |

### 3. questions（题目表）
| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| id | INTEGER | PRIMARY KEY | 主键ID |
| content | TEXT | NOT NULL | 题目内容 |
| question_type | ENUM | NOT NULL | 题型：single/multiple/judge/blank |
| difficulty | ENUM | NOT NULL | 难度：easy/medium/hard |
| options | TEXT | | 选项（JSON格式） |
| answer | VARCHAR(500) | NOT NULL | 正确答案 |
| explanation | TEXT | | 解析 |
| category_id | INTEGER | FOREIGN KEY | 所属科目ID |
| knowledge_point_id | INTEGER | FOREIGN KEY | 所属知识点ID |
| created_at | DATETIME | DEFAULT NOW | 创建时间 |
| updated_at | DATETIME | DEFAULT NOW | 更新时间 |

### 4. wrong_questions（错题表）
| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| id | INTEGER | PRIMARY KEY | 主键ID |
| question_id | INTEGER | FOREIGN KEY, NOT NULL | 题目ID |
| my_answer | VARCHAR(500) | | 用户答案 |
| error_count | INTEGER | DEFAULT 1 | 错误次数 |
| last_error_time | DATETIME | DEFAULT NOW | 最近错误时间 |
| is_marked | INTEGER | DEFAULT 0 | 是否标记 |
| created_at | DATETIME | DEFAULT NOW | 创建时间 |

### 5. practice_records（刷题记录表）
| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| id | INTEGER | PRIMARY KEY | 主键ID |
| question_id | INTEGER | FOREIGN KEY, NOT NULL | 题目ID |
| my_answer | VARCHAR(500) | | 用户答案 |
| is_correct | INTEGER | NOT NULL | 是否正确：0/1 |
| practice_time | DATETIME | DEFAULT NOW | 练习时间 |
| duration | INTEGER | | 答题时长（秒） |

## 表关系

```
categories (1) ──→ (N) knowledge_points
categories (1) ──→ (N) questions
knowledge_points (1) ──→ (N) questions
questions (1) ──→ (N) wrong_questions
questions (1) ──→ (N) practice_records
```
